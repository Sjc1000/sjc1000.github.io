About Me

Hi! My name is Steven, I live in Australia. I like to drink booze and write code!

I have been coding since I was 16, I started with a simple language called AutoHotkey, A language to easily bind hotkeys or make simple GUI programs.
Since then I've moved into languages like Python, tinkered with C/C++. Played with lua so I could make some nice conky configs for myself, learnt bash so I could operate my Linux computer better, Decent with HTML / CSS / JS, this website was made from scratch!

I usually hang out on IRC, where I've made some friends, we mainly discuss programming related topics and other useful things. I have been a helper in the AutoHotkey chat for years, Joined the Donation Coder chat and have participated in a few of their events, such as their New Apps For the New Year.

Thesedays I mainly use Python, creating simple programs that make my day to day life easier, from keeping an eye on certain YouTube users to downloading stuff at night while I sleep. If I need to automate it Python is where I go.
